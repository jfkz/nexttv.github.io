NEXT TV

Next TV - high-performance and robust OTT media streaming platform supporting multiple adaptive http streaming formats such as HLS, Microsoft Smooth Streaming and MPEG-DASH.

Formats

  • HLS
  • Smooth
  • MPEG-DASH

Codecs

  • H264/AVC
  • Advanced Audio Coding (AAC)
  • Dolby Digital AC-3

Playback

  • Timeshift
  • Archive
  • Rewind
  • Pause live tv (currently smooth only)
  • Multiple audio tracks
  • 5.1 audio

CDN integration

  • Level3
  • Akamai
  • Cdnvideo
  • Ngenix

Protection

  • DRM protection using Microsoft Playready (smooth)
  • AES-128 encryption for HLS protocol (Verimatrix in plans)
  • HAS url access control based on user ip
  • Protection from sharing HAS urls between users

High Availability

  • Next TV platform is distributed by design and tolerant to loss of any reasonable number of hardware nodes

Billing integration

  • Next TV provides simple REST API for HAS streams generation and statistics
