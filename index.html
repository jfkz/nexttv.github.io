NEXT TV

Next TV offer high-performance and robust self-hosted OTT platform streaming to almost any device using HLS, Microsoft Smooth Streaming and MPEG-DASH.

Multi-Formats

  • HLS
  • Smooth
  • MPEG-DASH

Codecs

  • H264/AVC
  • Advanced Audio Coding (AAC)
  • Dolby Digital AC-3

Playback

  • Timeshift
  • Archive
  • Rewind
  • Pause live tv (currently smooth only)
  • Multiple audio tracks
  • 5.1 audio

CDN integration

  • Level3
  • Akamai
  • Cdnvideo
  • Ngenix
  • Custom or private CDN could be added on demand.

Protection

  • DRM protection using Microsoft Playready (smooth)
  • AES-128 encryption for HLS protocol (Verimatrix in plans)
  • HAS url access control based on user ip
  • Protection from sharing HAS urls between users

High Availability

Next TV platform is distributed by design and tolerant to loss of any reasonable number of hardware nodes

Billing integration

Next TV provides simple REST API for HAS streams generation and statistics

Small Storage

Next TV OTT platform is designed to keep storage as small as possible so there is no duplication of data for multiple formats, DRMs and device dialects.

Customization

We are small company and can customise our products for your individual needs. New features by design can be easealy added on top of core functionality on demand.

Peer-to-Peer

We look forward to use P2P technologies for video distribution. This would be a must have feature when 4k video became mainstream - matter of a few years.

Deploy with Docker

Next TV platform is easily deployable to your private or public cloud infrastructure because we use docker as packaging and distribution tool.
